# Recruited_Reviewer_Snippet

Snippet from Feldman, E., Tosyali, A., & Overgoor, G. “An explainable AI framework for identifying fake review writers on
e-commerce platforms.” publication found here (https://papers.ssrn.com/sol3/papers.cfm?abstract_id=5156231)

The above code applies clustering algorithms to model predictions from a trained Graph Neural Network (GNN) used to detect fake (recruited) Amazon reviewers.
